## Objectif du projet

Ce projet a pour objectif de développer un système complet de gestion de location de véhicules en Python, en appliquant les principes fondamentaux de la OOP.  
L’application fonctionne en mode console et permet de gérer les clients, les véhicules, les réservations ainsi que la grille tarifaire associée.  

Le travail met l’accent sur :
- la conception modulaire d’un programme multi‑fichiers ;
- la séparation stricte des responsabilités (modèles, gestion des données, interface utilisateur, orchestration) ;
- l’utilisation de classes pour représenter les entités du domaine ;
- la sérialisation des données au format JSON ;
- la mise en œuvre d’un menu interactif permettant d’exécuter les fonctionnalités principales.
